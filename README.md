📚 Library Management System

📖 Overview

A web-based Library Management System developed to manage book records, purchases, and returns efficiently.

---

⚙️ Tech Stack

- Java
- JSP & Servlets
- MySQL
- HTML, CSS, JavaScript
- Apache Tomcat

---

🚀 Features

- 📘 Add, update, delete book details (CRUD)
- 🔍 View available books
- 🛒 Purchase / Issue books
- 🔄 Return books
- 📊 Manage book availability

---

🏗️ Modules

- Book Management
- Purchase / Issue Module
- Return Module

---

▶️ How to Run

1. Import project into Eclipse / IntelliJ
2. Configure MySQL database
3. Deploy on Apache Tomcat
4. Run in browser

---

💡 Learning Outcome

- CRUD operations using Java
- Database connectivity with MySQL
- JSP & Servlets for web development
- Basic system design understanding
